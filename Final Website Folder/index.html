Home Page

Our homepage is designed in a way that users can start using the app without creating an account.We designed our main page this way so that users feel unrestricted and inclined to explore around

Explore Page

Our Explore page displays a list of results based on the search query and hosts the results on the map. We took this idea from the popularity of the Yelp format, which displays restaurants and their relative locations on a map.

Company Page

This About screen is dedicated to providing our user with information about the organization. It includes tags that represent the skills that the student can develop in this organization as well as the location of the non-profit.
